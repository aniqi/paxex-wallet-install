# paxex-wallet-install
Install paxex-wallet (support Ubuntu 18.04, use at your own risk)

***commands:***

```
sudo apt update
sudo apt upgrade
sudo apt install wget
wget -q https://github.com/aniqi/paxex-wallet-install/releases/download/v1/install.sh && bash install.sh
```
